## About me

I'm a Full-Stack Developer transitioning into **AI Engineering**, currently enrolled in a Post-Graduate program in *Software Engineering with Applied AI* (Engenharia de Software em IA Aplicada).

My background is unusual for a developer: before writing production code, I spent years as a **CMO managing performance marketing campaigns** — Meta Ads, Google Ads, TikTok Ads — at scale. That means I understand both the technical and business sides of building AI-powered products.

Today I work as a **Developer at Detran-PA** (the Brazilian vehicle authority for Pará state), where I build institutional applications and see firsthand the opportunities for intelligent automation in government services.

I build with **TypeScript/JavaScript** as my primary stack for AI, using the APIs and frameworks that matter right now: Anthropic Claude, OpenAI, LangGraph.js, and the Model Context Protocol (MCP).
